ESG , el camino correcto para crear valor | Columnas

La sostenibilidad es clave para el sistema económico y hoy ha pasado a formar parte de las estrategias organizacionales para enfrentar la creciente desigualdad. El auge de iniciativas a nivel global como los Objetivos de Desarrollo Sostenible (ODS), el fortalecimiento de los requisitos de entidades reguladoras y la presión de clientes y trabajadores, son algunos de ellos.
