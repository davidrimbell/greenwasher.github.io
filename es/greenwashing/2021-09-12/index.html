Peón de Dama - El peligro de caer en el greenwashing
Las autoridades están estrechando el cerco sobre aquellas gestoras que podrían haber exagerado el uso de criterios de sostenibilidad para gestionar sus activos, lo que se conoce comogreenwashing. Este término anglosajón se utiliza para definir las malas prácticas que algunas empresas realizan cuando presentan un producto como respetuoso con el medio ambiente aunque realmente no lo es.
