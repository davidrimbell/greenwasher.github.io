Europa lanza la cruzada contra el greenwashing en las finanzas

Las autoridades europeas de supervisión de banca, seguros y mercados (EBA, Eiopa y ESMA) abren una nueva investigación para reunir información sobre las prácticas de lavado verde en estos sectores. Con este proyecto se ponen las bases para regular este tipo de conductas engañosas. Con este programa se pone las condiciones para que estas prácticas no se conviertan en delito.
